# File extension icons

Drop PNG icons in this folder using the lowercase file extension as the file name.

Examples:

- `py.png` is used for `.py` files.
- `md.png` is used for `.md` files.
- `json.png` is used for `.json` files.
- `default.png` is used when no extension-specific icon exists.

The app falls back to the platform file icon when neither the extension icon nor `default.png` exists.
